# Итерация и одновременное изменение коллекции

Симптом: **`ConcurrentModificationException`** при обходе `ArrayList`, `HashMap` и других обычных коллекций, если структура меняется «мимо» активного итератора.

## Когда использовать

- Удаляете элементы **во время цикла** по коллекции.
- Обходите `entrySet`/`keySet` у `Map` и вызываете **`remove`** на `Map` или на `Set` без итератора.

## Пошаговый подход

1. **Не** вызывайте `collection.remove(x)` внутри расширенного `for` по этой же коллекции.
2. Используйте **`Iterator.remove()`** после `next()`.
3. Или **`removeIf`** на коллекции.
4. Или соберите ключи/элементы для удаления и вызовите **`removeAll`** / **`keySet().removeAll`** один раз после цикла.

## Проверка результата

- Исключение не воспроизводится на типовом наборе данных.
- Сложность решения приемлема (избегайте квадратичных паттернов на больших `List` без нужды).
