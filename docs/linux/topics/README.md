# Linux Topics

Короткий список канонических тем Linux.

- [Basics](./basics/README.md) - базовая навигация, контекст и безопасная работа в терминале.
- [Files](./files/README.md) - раскладка каталогов (FHS), операции с файлами и каталогами, права и владельцы.
- [Processes](./processes/README.md) - поиск, мониторинг и корректная остановка процессов и сервисов.
- [Network](./network/README.md) - быстрая сетевая диагностика: host, port, DNS, routes.
- [Security](./security/README.md) - минимальные права, SSH-практики, обновления и аудит.
- [Troubleshooting](./troubleshooting/README.md) - системный разбор инцидентов и проверка гипотез.
