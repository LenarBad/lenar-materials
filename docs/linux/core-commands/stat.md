# stat

## Назначение

Показать **метаданные** объекта файловой системы: тип, права, владельца, размер, время изменения, номер inode и др. Удобно, когда `ls -l` недостаточно или нужен **точный** формат для скрипта.

## TLDR

Полный многострочный отчёт по пути.

```bash
stat /etc/hosts
```

Права (восьмеричные), владелец, группа, имя — одной строкой.

```bash
stat -c "%a %U:%G %n" /etc/myapp.conf
```

Размер файла в байтах.

```bash
stat -c "%s" /var/log/syslog
```

Права в «буквенном» виде, как в `ls`.

```bash
stat -c "%A" /bin/ls
```

Номер inode (диагностика жёстких ссылок).

```bash
stat -c "%i" /etc/hosts
```

Числовые uid и gid.

```bash
stat -c "%u %g" /etc/shadow
```

Время последнего изменения содержимого.

```bash
stat -c "%y" /var/log/nginx/access.log
```

Следовать по символической ссылке к целевому файлу.

```bash
stat -L /path/to/symlink
```

Сведения о файловой системе, на которой лежит путь (GNU: вместо метаданных одного inode).

```bash
stat --file-system /
```

## Синтаксис

```text
stat [опции] путь ...
```

## Поведение и параметры

**Без `-c`** на типичной Linux-системе выводится многострочный отчёт: файловая система, inode, права в буквенном и восьмеричном виде, ссылки, uid/gid, размер, три времени (доступа, модификации, изменения inode) и имя.

**`-c формат`** (GNU coreutils) — задать **одну строку** вывода; в строке используют **спецификаторы**, начинающиеся с `%`.

| Спецификатор | Смысл (типично) |
| --- | --- |
| `%n` | Имя файла |
| `%s` | Размер в байтах |
| `%a` | Права в восьмеричном виде |
| `%A` | Права в буквенном виде (`-rw-r--r--`) |
| `%U`, `%G` | Имя пользователя и группы-владельца |
| `%u`, `%g` | Числовые uid/gid |
| `%i` | Номер inode |
| `%y` | Время модификации содержимого |

Полный список и точное поведение — в `man stat` на вашей машине; набор спецификаторов может немного отличаться между GNU и BSD.

| Параметр | Что делает |
| --- | --- |
| `-f` | Показать информацию о **файловой системе**, в которой лежит путь (отдельный режим; детали в man) |
| `-L` | Следовать по символической ссылке к цели |
| `-t` | Краткий однострочный вывод (формат зависит от реализации) |

## Примеры

Сравнить права двух файлов.

```bash
stat -c "%a %n" /etc/passwd /etc/shadow
```

Узнать время последнего изменения в удобном виде.

```bash
stat -c "%y" /var/log/nginx/access.log
```
