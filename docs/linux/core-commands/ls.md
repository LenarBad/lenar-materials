# ls

## Назначение

Показать **имена файлов и каталогов**: в текущем каталоге или в том, который вы укажете. С ключами — ещё и права, владельца, размер, время изменения.

## TLDR

Краткий список имён в текущем каталоге.

```bash
ls
```

Подробно: права, владелец, размер; скрытые имена; размеры «по-человечески».

```bash
ls -lah
```

Содержимое другого каталога.

```bash
ls -lah /var/log
```

По одному имени в строку (удобно для `wc` и циклов).

```bash
ls -1
```

Сортировка по времени изменения, новые сверху.

```bash
ls -lt
```

Показать сам каталог, а не то, что внутри.

```bash
ls -ld /etc
```

Скрытые файлы без лишних `.` и `..` (GNU).

```bash
ls -A
```

Сортировка по размеру (крупные внизу или вверху — зависит от версии; уточните `man ls`).

```bash
ls -lahS
```

Предсказуемый порядок имён (полезно в скриптах).

```bash
LC_ALL=C ls -1
```

## Синтаксис

```text
ls [опции] [файл или каталог] ...
```

На Linux чаще всего это программа из **coreutils**. На macOS стоит другая реализация — часть ключей и поведение могут отличаться; ориентируйтесь на `man ls` на той машине, где работаете.

## Поведение и параметры

**Без ключей** — обычно краткий список имён в несколько колонок (как именно — зависит от оболочки и локали).

**`-l`** включает «длинный» формат: для каждого объекта — права, число ссылок, владелец, группа, размер, время, имя.

| Параметр | Что делает |
| --- | --- |
| `-l` | Подробный список |
| `-a` | Показать и скрытые имена (начинающиеся с точки), вместе с `.` и `..` |
| `-h` | Вместе с `-l`: размеры в удобных единицах (КиБ, МиБ и т.д.) |
| `-d` | Если указан каталог — показать его самого, а не содержимое |
| `-t` | Сортировать по времени изменения (новые сверху) |
| `-S` | Сортировать по размеру |
| `-R` | Зайти в подкаталоги рекурсивно |
| `-1` | По одному имени в строку |
| `--color=auto` | Раскраска типов файлов (если поддерживается) |

На GNU-версии часто полезны ещё `-A` (скрытые файлы, но без `.` и `..`) и `-i` (номер inode).

**Большие каталоги** — вывод может быть медленным; иногда проще искать через `find`. **Сортировка имён** в скриптах может зависеть от языка окружения; для предсказуемого порядка иногда задают локаль, например `LC_ALL=C ls`.

## Примеры

Показать сам каталог `/etc`, а не его содержимое.

```bash
ls -ld /etc
```

Недавно изменённые файлы в каталоге загрузок (первые строки).

```bash
ls -lt ~/Downloads | head
```
