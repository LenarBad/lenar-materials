# free

## Назначение

Показать использование **оперативной памяти и swap**: общий объём, занято, свободно, буферы/кэш (в зависимости от колонок). Быстрый снимок при подозрении на нехватку RAM или давление по swap.

## TLDR

Человекочитаемые величины.

```bash
free -h
```

Вывод в мегабайтах (без суффиксов `-h`).

```bash
free -m
```

Непрерывное обновление каждые 2 секунды.

```bash
free -hs 2
```

Широкий формат с дополнительными колонками (если поддерживается).

```bash
free -hw
```

## Синтаксис

```text
free [опции]
```

Интерпретация строк **Mem** и **Swap** зависит от ядра и версии `free`; на новых системах полезно сверяться с `man free`.

## Поведение и параметры

**«Свободная»** память в Linux часто мала — ядро использует остаток под **кэш**; важнее смотреть на **доступность** для приложений и рост **swap**, а не одну цифру «free».

| Параметр | Что делает |
| --- | --- |
| `-h`, `--human` | Суффиксы Ki/Mi/Gi |
| `-m` / `-g` | Мегабайты / гигабайты |
| `-s N` | Повторять каждые N секунд |
| `-w` | Широкий формат |
| `-t` | Строка итогов |

Для постоянного мониторинга нагрузки процессов обычно удобнее **`top`** / **`htop`**.

## Примеры

Один снимок перед тикетом о OOM.

```bash
free -h && ps -eo pid,cmd,%mem --sort=-%mem | head
```
