# Tips & Tricks

<div class="lenar-tip-grid" markdown="block">

!!! note "Очистка экрана и «сломанный» терминал"

    - Очистить вывод на экране: `clear` или **Ctrl+L** (часто не стирает буфер прокрутки, только сдвигает приглашение вверх).
    - Если после вывода бинарного файла «поехали» символы и цвета, помогает сброс терминала:

    ```bash
    reset
    ```

!!! note "История команд"

    - Список недавних команд:

    ```bash
    history
    ```

    - Поиск по истории:

    ```bash
    history | grep часть_команды
    ```

    - **Ctrl+R** — интерактивный поиск по истории (в большинстве интерактивных оболочек).

!!! note "Цепочки команд"

    | Запись | Смысл |
    | --- | --- |
    | `к1 ; к2 ; к3` | Выполнить по очереди, даже если предыдущая команда завершилась с ошибкой |
    | `к1 && к2` | `к2` только если `к1` успешна (код выхода 0) |
    | `к1 \|\| к2` | `к2` только если `к1` завершилась с ошибкой |

!!! note "Справка по синтаксису"

    - Полное руководство: `man имя_команды`
    - Краткая справка: `имя_команды --help`
    - Короткие примеры (если установлен клиент **tldr**, пакет обычно называется `tldr`): `tldr имя_команды`

!!! note "Перенаправления"

    | Оператор | Назначение |
    | --- | --- |
    | `>` | Вывод в файл (файл перезаписывается) |
    | `>>` | Добавить вывод в конец файла |
    | `<` | Взять стандартный ввод из файла |
    | `2>` | Направить **поток ошибок** в файл (часто вместе с `>` для раздельного лога) |

    Пример: стандартный вывод и ошибки в разные файлы.

    ```bash
    rm -vf папка файл > out.txt 2> err.txt
    ```

!!! note "Редактирование текущей строки (режим по умолчанию, как в emacs)"

    Удобно в длинных командах и при правке пути.

    | Клавиши | Действие |
    | --- | --- |
    | **Ctrl+A** | К началу строки |
    | **Ctrl+E** | К концу строки |
    | **Ctrl+U** | Удалить всё до курсора |
    | **Ctrl+K** | Удалить от курсора до конца строки |
    | **Ctrl+W** | Удалить слово слева от курсора |
    | **Ctrl+Y** | Вставить последнее «вырезанное» (paste) |
    | **Ctrl+_** | Отмена последнего изменения в строке |

    В **zsh** для перехода по словам часто используют **Alt+B** / **Alt+F** (зависит от терминала и настроек).

!!! note "Дополнение по Tab"

    - Введите начало имени файла или команды и нажмите **Tab** — оболочка допишет или покажет варианты (**Tab** дважды).
    - Сокращает опечатки в длинных путях и именах.

!!! note "Алиасы"

    Временный алиас (до выхода из сессии):

    ```bash
    alias имя='команда с аргументами'
    ```

    Постоянно — добавьте ту же строку в конец профиля оболочки:

    | Оболочка | Файл (типично) |
    | --- | --- |
    | bash | `~/.bashrc` |
    | zsh | `~/.zshrc` |
    | fish | `~/.config/fish/config.fish` |

    Узнать текущую оболочку: `echo $0` или `echo $SHELL`.

!!! note "Фон, приостановка, долгие задачи"

    - **Ctrl+Z** — приостановить текущую задачу в оболочке.
    - `bg` — продолжить в **фоне**, `fg` — вернуть на передний план.
    - `jobs` — список задач сессии.
    - Запуск, переживающий выход из сессии (грубо):

    ```bash
    nohup долгая_команда &
    ```

    Для нескольких сессий и отсоединения от SSH удобнее **tmux** или **screen**.

!!! note "tmux (минимум)"

    ```bash
    tmux new -s имя_сессии
    ```

    Отсоединиться: **Ctrl+B**, затем **D**. Подключиться снова:

    ```bash
    tmux attach -t имя_сессии
    ```

!!! note "Планирование по расписанию (cron)"

    Редактор расписания для пользователя:

    ```bash
    crontab -e
    ```

    Формат строки: минута, час, день месяца, месяц, день недели, затем команда. Пример — каждый день в 02:00:

    ```text
    0 2 * * * /home/user/backup.sh
    ```

    Подробнее о правах на файлы и `chmod` см. тему [Files](../topics/files/README.md) и шпаргалки команд.
